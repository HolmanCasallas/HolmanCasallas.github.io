Práctica De Graffiti

Materiales

1. Pintura en aerosol.

2. Marcadores.

3. Plantillas.

4. Guantes.

5. Gafas de seguridad.

6. Pañuelos o trapos.

7. Lona o papel.

8. Cámara.

9. Máscara antigás.

Paso A Paso

1. Tener una idea clara del diseño que quieres realizar

2. Preparar la superficie, asegurar que donde se va a realizar este limpia y libre de polvo si no está limpia, limpia la superficie con un trapo húmedo o una brocha.

3. Aplica la base utiliza pintura en aerosol para crear la base de tu diseño. Puedes usar un color sólido o un degradado.

4. Agrega detalles si quieres utiliza marcadores o pintura en aerosol para agregar detalles a tu diseño. Empieza con los detalles más grandes y luego ve agregando detalles más finos.

5. Agrega colores y texturas utiliza marcadores o pintura en aerosol para agregar colores y texturas a tu diseño. Puedes usar diferentes técnicas, como el sombreado o el degradado.

6. Refina detalles: Utiliza marcadores o pintura en aerosol para refinar los detalles de tu diseño.

7. Agrega sombras y luces sigues utilizando de tu preferencia marcadores o pintura en aerosol para agregar sombras y luces a tu diseño. Esto ayudará a dar profundidad y dimensión a tu grafiti.

8. Revisa y retoca revisa tu grafiti y retoca cualquier error o área que necesite mejora.

9. Firma tu obra agrega tu firma o tag a tu grafiti.

Práctica De Dibujo

Materiales

1. Papel.

2. Lápices.

3. Marcadores.

4. Gises y tisas.

Paso A Paso

1. Preparar el espacio donde se va a realizar el trabajo

2. Elige el papel con el que más te sientas más cómodo para realizar este trabajo.

3. Haz una base para poder guiarte qué forma va a tener tu dibujo.

4. Con un lápiz empiezas a realizar detalles finos.

5. Luego utiliza un lápiz o un marcador para sombrear tu dibujo.

6. Agrega texturas: Utiliza un lápiz o un marcador para agregar texturas a tu dibujo. Puedes utilizar diferentes técnicas, como el puntillismo o el trazado de líneas.

7. Revisa y retoca: Revisa tu dibujo y retoca cualquier error o área que necesite mejora.

8. Colorea (opcional): Si deseas agregar color a tu dibujo, utiliza marcadores de colores o pinturas.

Práctica De Pintura

Materiales

1. Lienzo o papel: Lienzo de algodón o papel de acuarela.

2. Pinturas: Pinturas al óleo, acrílicas o acuarelas.

3. Pinceles: Pinceles de cerda, pinceles de nylon o pinceles de espuma.

4. Paleta: Una paleta de madera o plástico para mezclar las pinturas.

5. Agua: Agua para mezclar con las pinturas y limpiar los pinceles.

6. Solventes: Solventes como el turpentine o el acetato para limpiar los pinceles

7. Esmalte: Esmalte o barniz para proteger y dar brillo a la pintura.

Paso A Paso

1. Prepara tu espacio de trabajo: Asegúrate de que tu espacio de trabajo esté limpio, ordenado y bien iluminado. Cubre tu mesa o superficie de trabajo con un paño o papel para protegerla de las pinturas.

2. Elige tu lienzo o papel cuando elijas alguno de estos papeles dependiendo con cual te sientas más cómodo asegúrate de que esté limpio y seco.

3. Haz un bosquejo utiliza un lápiz o un marcador para hacer un bosquejo de la pintura que deseas realizar. Comienza con las líneas básicas y luego agrega detalles.

4. Mezcla tus pinturas, utiliza tu paleta para mezclar las pinturas inicia con los colores primarios y mézclalos para crear los colores que necesites.

5.Luego pinta el fondo utilizando un pincel para pintar el fondo de tu lienzo o papel. Puedes utilizar un color sólido o un degradado.

6.Despues agrega detalles con un pincel para agregar detalles a tu pintura. Iniciando desde los más grandes hasta terminar en los más finos.

7. Agrega texturas: Para agregarle texturas utiliza un pincel o un instrumento de textura para agregar texturas a tu pintura también puedes utilizar diferentes técnicas, como el impasto o el scraping.

8. En este paso revisa tu pintura y retoca cualquier error o área que necesite mejora.

9. Y ya por último se agrega el barniz cuando este seca la pintura para protegerla y que le de brillo.

Práctica de Personalización

Materiales

1. Cautín o taladro.

2. Quita pintura especial.

3. Lija o pulidor.

4. Pintura.

5. Pegatinas o pintura acrílica.

6. Fijador.

Paso A Paso

1. toma el carrito y lo destapas con el cautín o el taladro, toca tener mucha precaución al usar alguno se estos dos elementos.

2. luego se sumerge o se le aplica el quitador de pintura y se deja actuar por 10 minutos.

3. se quita el exceso de pintura y se procede a lijarlo y pulirlo para que a la pintura no quede con algún grumo.

4. se le aplica la pintura y se deja secar dependiendo el color de tu preferencia.

5. cuando la pintura esta lista, se procede a aplicar los detalles como las farolas o los estampados se pueden realizar con pintura acrílica o para más facilidad con pegatinas especiales.

6. y el último paso es aplicarle un fijador para resaltar su brillo y dejarlo secar.

7. En este paso ya es la montura del carrito y listo ya quedaría el carrito listo.
